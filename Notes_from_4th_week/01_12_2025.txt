************** Function calling with OpenAI Responses API *************
- Function calling is the way for OpenAI model to interface with external system and
access the data outside their training data

- Flow
a. Send initial request to model
b. Model respond with tool invocation (Specifying which tool to call and with wich env)
c. Application will execute the requested toll on the basis of input parameters
d. Return tool result to model
e. Receive the final response, which may be the completion result or additional tool calls.
